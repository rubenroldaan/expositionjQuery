Drag&Drop (Elementos arrastrables)

Gracias a la librería jQuery UI existe la posibilidad de que los elementos en una página web puedan ser arrastrados por el usuario mediante el ratón. Esto se lleva a cabo mediante el método draggable().

Su sintaxis sería:

$("elemento").draggable({opcion/evento});

Existen una serie de opciones y eventos para personalizar el arrastre:

  • axis: Permite mover el elemento sobre un solo eje.
  • delay: Retrasa en milisegundos el movimiento del elemento.
  • containment: No permite que el elemento sobresalga del elemento padre.
  • handle: Indicamos desde que descendiente del elemento hacemos arrastrable el elemento.
  • cancel: Hace al elemento no arrastrable. Puede aplicarse a sus hijos.
  • disabled: Similar a cancel, usando true o false.
  • opacity: Se puede cambiar la opacidad del elemento mientras es arrastrado.
  • revert: Al soltar el elemento vuelve a su posición inicial.
  • cursor: Permite cambiar el cursor mientras se arrastra el elemento. Para conocerlos más a fondo haz click aquí.

Eventos

Existen eventos a los que asignar métodos a elementos draggables.

  • start: Cuando empiezas a mover el elemento
  • stop: Cuando sueltas el elemento.
