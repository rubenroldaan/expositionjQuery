¿Qué es jQuery?

jQuery es una librería de JavaScript cuya función es parecida al lenguaje mencionado, interactuar con los documentos HTML dinámicamente pero de una manera más simplificada. jQuery es un software de código abierto que lo que consigue es acortar el código necesario para realizar una acción en concreto (manipular el árbol DOM, crear animaciones, manejar eventos) e incluso tiene interacción con AJAX para realizar peticiones a una base de datos.

Así sería un código en JavaScript

e = document.getElementById("id")
e.style.fontSize="15px";

Y así se simplificaría en jQuery

$("#id").css("font-size","2rem");

Características

  • Es un software libre, por lo que puede ser empleado por cualquier usuario de manera gratuita.
  • Su librería permite actualizaciones constantes y rápidas.
  • Posee un código abierto y compatible con diferentes navegadores.
  • Es fácil de usar, por lo que permite ahorrar tiempo y esfuerzo.
  • Su desempeño se integra muy bien con AJAX, una técnica de desarrollo web.
  • Permite realizar animaciones, efectos y personalizaciones.
  • Es compatible con diferentes buscadores como Google Chrome, Microsoft Edge, Firefox, IE, Safari, Android e IOS, cuyas páginas web se deben programar en formas diferentes.
