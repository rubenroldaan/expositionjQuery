Uno de los puntos fuertes de jQuery es la capacidad de establecer efectos a un elemento para crear dinamismo en una página web.

El método animate()

El método animate() es el más potente para generar animaciones. Con él podemos cambiar de manera dinámica cualquier atributo css. Su sintaxis sería:

     .animate( propiedades [, duracion ] [, easing ] [, callback ] )

  • propiedades: Los cambios en el CSS que se van a representar.
  • duracion: Cuánto va a durar la animación en milisegundos. También admite los valores "slow" y "fast".
  • easing: Forma en la que se va a producir la transición, están basadas en las curvas de Bezier.
  • callback:Función que se ejecuta cuando la animación haya concluido con éxito.

La familia fade()

Los métodos fade manipulan la propiedad opacity de CSS. Existen 4 métodos:

  • fadeIn(x): Sube la opacidad del elemento, haciéndolo visible.
  • fadeOut(x): Baja la opacidad del elemento, haciéndolo oculto.
  • fadeToggle(x): Sube y baja la opacidad del elemento alternativamente.
  • fadeTo(x): Baja la opacidad del elemento pero no termina oculto.

show, hide y toggle

Estos métodos se encargan de la aparición y desaparición de un elemento.

  • show(): Muestra el elemento oculto.
  • hide(): Oculta el elemento si es visible.
  • toggle(): Muestra u oculta el elemento en función de si es visible o no.

slide

Los métodos slide manipulan la altura (height) del elemento.

  • slideDown(): Despliega el elemento.
  • slideUp(): Recoge el elemento.
  • slideToggle(): Recoge y despliega el elemento.

Manejador de clases

Existen métodos que alteran las clases de los elementos.

  • addClass(): Añade al elemento una clase.
  • removeClass(): Quita al elemento una clase.
  • toggleClass(): Añade y quita al elemento una clase.
class solido {
     border: 3px solid red;
}

class solido {
     border: 5px dotted green;
     border-radius: 6px;
}


jQuery UI

Es una librería con algunos componentes para crear más dinamismo a partir de los métodos básicos de efectos de jQuery. Primero deberemos descargar la librería desde la página oficial y a partir de ahí elegiremos los componentes. Se utiliza con los métodos show, hide y toggle.

Su sintaxis sería:

$("elemento").metodo( "tipo_efecto" , "velociad_efecto" );
