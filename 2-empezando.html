Estructura de jQuery

Siempre que vayamos a hacer uso de la librería jQuery, deberemos utilizar una sintaxis estructurada, divida en fragmentos fácilmente diferenciables:

  • Función jQuery
  • Un selector como parámentro
  • Uno o varios métodos de la librería
  • Opcionalmente, parámetros del método.
<script type="text/javascript">
$(document).ready(function() { 
     $("p").css("background-color","red");
     jQuery("p").css("background-color","red");

     $("input").attr("value","25");
     jQuery("input").attr("value","25");
});

Puedes ver que usar $ o usar la función jQuery es lo mismo.


De este modo, la estructura de jQuery sería así:

     $(nodo_seleccionado).nombre_del_metodo([parámetros])


$(document).ready(), ¿por qué?

Cuando veas código jQuery en cualquier sitio verás que siempre se encuentra esta sentencia al principio del código y que rodea todo lo que se declara en jQuery. Básicamente el método ready() se encarga de que, una vez esté cargada la página y se haya generado el árbol DOM, todo lo de dentro del código pueda ejecutarse ya que, como vimos anteriormente, jQuery trabaja directamente sobre los nodos del árbol DOM.

Este sería el código que habría que escribir:

$(document).ready(function() {
     // Aquí iría el código del script
});
